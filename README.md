## 运行说明
### 1. 准备
  	1). 请确保安装了node环境 查看是否已经安装: $node -v  
  	2). 请确保安装了mongodb, 并启动了对应的服务 启动：$mongo
 
### 2. 启动后台应用
	1). 进入应用
	2). 执行命令: `npm start`
	3). 提示连接成功
